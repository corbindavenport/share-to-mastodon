Mercury Settings

This is where you can add or remove saved Mastodon servers or change the keyboard shortcut. Your changes are saved automatically.

Bluesky New
This will add Bluesky to the share list. Only the default bsky.app web app is supported.
Mastodon and Fediverse
This must be a server running Mastodon, Misskey, or another compatible social media platform. You can also enter elk.zone for the Elk web app or bsky.app for Bluesky.
Other settings
Open about:addons in a new tab, click the settings gear button, then select "Manage extension shortcuts."
